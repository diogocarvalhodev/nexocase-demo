# Roadmap Curto (30-60-90 dias)

## 30 dias
- Adicionar testes automatizados de fumaça para login, refresh, incidentes e health
- Publicar dashboard técnico com métricas de erros, latência e throughput
- Criar script de demo guiada para entrevista técnica e comercial

## 60 dias
- Introduzir pipeline CI com lint, testes e verificações de segurança
- Adicionar trilha de auditoria exportável para revisão de compliance
- Melhorar cobertura de validação de dados e contratos de API

## 90 dias
- Preparar deploy cloud com ambiente staging e observabilidade centralizada
- Definir SLOs de API e política de incident response interno
- Evoluir integrações externas (alerting/notificação) sem comprometer isolamento por tenant

## Critérios de sucesso
- Tempo médio de setup local abaixo de 10 minutos
- Demo completa executada em 90 segundos sem intervenções manuais
- Evidências técnicas suficientes para aprovação em triagem backend/full stack
